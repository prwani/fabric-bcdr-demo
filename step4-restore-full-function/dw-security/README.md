# Warehouse and workspace security replay

Use the output generated by the Step 0 backup scripts to restore security in the recovered environment.

## Required upstream artifacts

- generated output from `ScriptFabricDWSecurity.sql`
- generated output from `ScriptWorkspacePermissions.ps1`

## Restore order

1. Restore or recreate the target workspace and warehouse artifacts.
2. Apply warehouse security replay scripts.
3. Apply workspace role replay scripts.
4. Validate access with representative personas.
